# Spotify-Song-Recommender

## Setup
Create virtual environment:
```sh
conda create -n spotify-env python=3.11
```

Activate the environment: 
```sh
conda activate spotify-env 
```
Install Packages:
```sh
pip install -r requirements.txt
```

## Usage

Run the script:
```sh
python -m app.songrecommender 
```

## Testing
Run Tests:
```sh
pytest
```
